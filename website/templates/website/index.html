Tentang Saya

Halo, saya Abdurrahman. Sejak kecil, saya selalu memiliki ketertarikan yang besar terhadap mekanisme dunia teknologi. Saat ini, sebagai mahasiswa Teknik Telekomunikasi angkatan 2024, fokus utama saya berpusat pada eksplorasi komunikasi data—khususnya pemahaman mendalam pada lower layer OSI (Layer 2, 3, dan 4) menggunakan instrumen seperti Cisco, Mininet, dan Ryu Controller. Selain antusias membedah arsitektur jaringan, saya juga mengeksplorasi implementasi machine learning dan pengolahan data menggunakan Python. Ruang digital ini didedikasikan sebagai etalase untuk membagikan minat, mendokumentasikan perjalanan, serta menampilkan berbagai hasil proyek independen saya.
